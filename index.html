El Mundo Medieval

Ambientación

Un universo donde caballeros, magos y criaturas digitales luchan en épicas batallas. Cada carta es fragmento de código que se transforma en un hechizo o habilidad.

Progresión

  • Aprendiz de Código → Ganas experiencia y desbloqueas cartas
  • Desbloquear cartas complejas y combinaciones avanzadas
  • Rango final: Arquitecto del Código

Mecánicas principales

1
Creación de Cartas

Seleccionás una "mica" y la programás con plantillas o código simplificado. El sistema limita el poder para garantizar equilibrio.

2
Fases del duelo

Colocación (boca abajo) → Revelación y resolución de hechizos → Combate principal.

3
Tipos de cartas

Cartas Principales, Hechizos (bonificaciones, penalizaciones, destrucción) y cartas especiales según personaje.

Cartas & ejemplos

Guardián de Acero (Tanque)

class Guardian:
    def __init__(self, hp=150, armor=10):
        self.health = hp
        self.defense_rating = armor

    def absorb_damage(self, incoming_damage):
        damage_taken = max(0, incoming_damage - self.defense_rating)
        self.health -= damage_taken

Alta salud y capacidad de absorber daño para proteger otras unidades.
